# ENAMTILA.space — House of Life

A global sanctuary for spiritual practitioners, artists, and cultural catalysts.

## About

ENAMTILA.space is a decentralized network supporting creators and healers who refuse to be erased by borders, capitalism, or institutional gatekeeping. Named after the Sumerian "House of Life," we build infrastructure for practitioners the world tries to silence.

## Website Structure

- **Home** (`index.html`) - Main landing page with hero, manifesto, and overview
- **Gallery** (`gallery.html`) - Films, photography, painting, digital art, and mixed media
- **Practitioners** (`practitioners.html`) - Spiritual coaches, healers, psychics, writers, artists, speakers
- **Tents** (`tents.html`) - Our 12 global sanctuary spaces across continents
- **Marketplace** (`marketplace.html`) - Original artworks, prints, healing sessions, commissions
- **Community** (`community.html`) - Join the collective, events, ways to participate
- **Stay With Us** (`couchsurfing.html`) - Artist couchsurfing network

## Design

- **Color Palette**: Ancient earth tones (clay, lapis, gold) meets cosmic colors
- **Typography**: Cormorant Garamond (display), Outfit (body), Noto Kufi Arabic
- **Features**: Animated starfield, smooth scroll reveals, responsive design
- **Philosophy**: Sumerian sanctuary meets cosmic rebellion

## Deployment

The site is deployed via GitHub Pages from the `/docs` folder.
